= The GET Method
:page-aliases: http-get-method.adoc

== Synopsis

`GET` requests the transfer of a target resource in a selected representation from the server to the client.

== Examples

.HTTP 1.1 Request to retrieve a given resource
----
GET /customers/9839 HTTP/1.1
Host: amce.co.uk
User-Agent: curl/8.0.0
Accept: */*
----

.HTTP/3 Request to retrieve a given resource
----
:method: GET
:scheme: https
:authority: amce.co.uk
:path: /customers/9839
user-agent: curl/8.0.0
accept: */*
----
