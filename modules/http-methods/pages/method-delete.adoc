= The DELETE Method
:page-aliases: http-delete-method.adoc

== Synopsis

`GET` requests
the server to remove a resource.

== Examples

.HTTP 1.1 Request to remove a given resource
----
DELETE /customers/9839 HTTP/1.1
Host: amce.co.uk
User-Agent: curl/8.0.0
Accept: */*
----

.HTTP/3 Request to remove a given resource
----
:method: DELETE
:scheme: https
:authority: amce.co.uk
:path: /customers/9839
user-agent: curl/8.0.0
accept: */*
----
