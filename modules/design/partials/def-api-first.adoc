= API First
endif::[]

*API First* ist ein Konzept zur Beschreibung und Veröffentlichung der Operationen eines Systems,
wofür für die Schnittstellen des Systems eine formale Spezifikation erstellt und zwischen allen beteiligten Parteien abgestimmt und geteilt wird.

Damit einher geht das Ziel die *Nutzung und Implementation* einer API voneinander *zu entkoppeln*, da API-Clients nur die API Spezifikation kennen müssen, jedoch nicht die technischen Details deren Implementierung auf Seiten des API-Providers.
