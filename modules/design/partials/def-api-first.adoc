= API First
endif::[]

Bei dem Konzept *API Frist* wird für eine Schnittstelle eine formale Spezifikation erstellt und zwischen allen beteiligten Parteien abgestimmt und geteilt.

Die Spezifikation dient als Grundlage für jede Implementierung der Schnittstelle.
Handelt es sich um eine formale Spezifikation, kann diese für die Generierung von Quellcode und weiteren Artefakten genutzt werden.

Ziel ist es, eine technologie- und implementierungsunabhängige Beschreibung einer Schnittstelle zu erstellen,  die ohne Abhängigkeiten zu einer bestimmten Implementierung zur Verfügung gestellt werden kann.
